¿Necesitas ayuda?

Estamos aquí para ayudarte con cualquier duda o problema que puedas tener al usar el Sistema de Útiles Escolares. Nuestro equipo de soporte técnico está disponible para asistirte.

¿Cuándo contactarnos?

No dudes en escribirnos si experimentas:

  • Problemas para iniciar sesión
  • Errores al crear o editar listas de útiles
  • Dificultades para acceder a ciertas funciones
  • Problemas técnicos o errores del sistema
  • Dudas sobre el uso de alguna función
  • Necesidad de cambiar información de perfil
Información a incluir en tu consulta

Para poder ayudarte de manera más eficiente, incluye en tu mensaje:

  • Tu nombre completo y rol en el sistema
  • Descripción detallada del problema
  • Pasos que realizaste antes del error
  • Capturas de pantalla si es posible
  • Navegador web que estás utilizando
Horario de Atención

Lunes a Viernes: 7:00 AM - 5:00 PM (Hora de Costa Rica)
